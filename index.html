📝 Writing a Post

Create a markdown file in posts/<slug>.md and add its metadata to posts/manifest.json. Commit & push — Vercel auto-deploys.

🔍 Filtering

Click a month chip to show only posts from that month. Click "All" to reset. Filter is saved in your browser.

↕ Sorting

Click ↓ Newest or ↑ Oldest to toggle sort direction. Your preference is saved.

📄 Export PDF

Click the PDF button on the home page or Export PDF on a post page, then choose "Save as PDF" in the print dialog.

🎨 Theme

Toggle between dark and light mode using the 🌙/☀️ button in the header.

🔗 Links

Each post supports GitHub links, Vercel links, YouTube embeds, images, code blocks, and more. Just use standard markdown.
